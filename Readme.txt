001_PyQt_Test.py script and the modules found here are meant for a practice/learning exercise in creating GUI applications in Python using PyQt5 and Matplotlib

This script runs a "Teacup Temperature Data Visualizer" application to collect and plot temperature data in a tea cup across time.
The user can manually enter a set of temperature data and save it into a csv file and then plot that temperature data and overlay against other similar data sets

Application_Screenshot.png file shows an example of the application window

With this project I learned the following concepts:

1. Basic understanding of widgets in PyQt5
2. Grouping and layouts to achieve the desired positioning of various GUI elements
3. Embedding Matplotlib plots and drawing graphs in PyQt5 windows
4. Reading and writing to csv files using Pandas
5. Processing time data in pandas data frames
